NewGRFs:

Standardised Expanded Track Set SETS

SETS is the largest trackset avaliable for OpenTTD with a more than 64 railtypes. As such it has to be split into two separate newgrfs one for narrow gauge and standard guage respectively, with even more potentially on the table. Get it on Bananas or read more here.

ScaNDinavian Train Set (SNDTS) (Alpha)

SNDTS is an under development replacment for Swedish Trains, Danish Trains and the Norwegian (NSB) Trainset. Developed jointly by Me, BL, dP, and Meja. I am mostly working on swedish 892 mm trains at the moment but that will probably change at some point. Early alpha avaliable on Bananas, code avaliable on Github.

French Trains Set (Narrow Gauge) Spoofer

French Trains Set (Narrow Gauge) Spoofer is a little grf I made when I found out French Trains Set (Narrow Gauge) doesn't work with any trackset but instead comes with a whitelist of supported sets. This newgrf pretends to be the accompanying trackset so that French Trains Set (Narrow Gauge) will work with any trackset. Download it here.

Chips style dock objects

Chips style dock objects was made to make building large port complexes easier with the added benefit of matching nearby firs industries. Downloads are on Bananas and the code is on Github. Make sure to check out ISR/DWE-style Objects II too.

Trainset overrides

I have also made 3 newgrfs that override other trainset newgrfs, fixing them and/or adding Support for multi-voltage tracks so that they work with SETS. These are Swedish trains fixed, Danish Trains Voltage Override and DACH Multivoltage Override, all are avaliable on Bananas.

Flowers and Road runway

Flowers and Road runway are two small newgrfs I made, Flowers replaces the rough ground with mushrooms and flowers but only works with the OpenGFX. Road runway is a small airport without the runway tiles allowing you to place anything in their place, be it a road or something else. Both are avaliable on Bananas
